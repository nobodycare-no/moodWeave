# MoodWeave ✦

> 一个轻量、优雅的在线灵感情绪板（Mood Board）工具。

拖拽图片和文字卡片，快速拼贴你的视觉灵感，一键导出分享。

## ✨ 特性

- 🎨 **暗色潮酷视觉** — 深色背景 + 霓虹点缀，灵感创作更沉浸
- 🖼️ **自由画布** — 缩放、平移、拖拽，随心所欲
- 📦 **多 Board 管理** — 同时管理多个灵感项目
- 💾 **本地存储** — 数据保存在浏览器，无需注册登录
- 📤 **一键导出** — 将情绪板导出为 PNG 图片

## 🚀 快速开始

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173` 即可使用。

## 🛠️ 技术栈

- Vue 3 (Composition API + `<script setup>`)
- TypeScript
- Vite
- localStorage
